# Fonts Ordner

## Tropical Island Schriftart

Die Schriftart "Tropical Island" von scratchones muss hier abgelegt werden.

**Download-Link:** Die Schriftart kann von der scratchones Website heruntergeladen werden (Comic Cartoon, Überschrift "Tropical Island", 605 downloads).

**Dateinamen:** 
- TropicalIsland.ttf
- TropicalIsland.otf
- TropicalIsland.woff
- TropicalIsland.woff2

Legen Sie die Schriftart-Datei(en) in diesen Ordner ab. Das System unterstützt alle gängigen Font-Formate (TTF, OTF, WOFF, WOFF2).
