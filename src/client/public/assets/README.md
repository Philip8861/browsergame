# Assets-Verzeichnis

## Login-Hintergrundbild

Um das tropische Strand-Bild als Login-Hintergrund zu verwenden:

1. **Bild hinzufügen:**
   - Benenne das Bild: `login-background.png`
   - Speichere es in diesem Verzeichnis: `src/client/assets/login-background.png`

2. **Unterstützte Formate:**
   - `.png` (aktuell verwendet)
   - `.jpg` / `.jpeg`
   - `.webp`

3. **Bildgröße:**
   - Empfohlene Auflösung: 1920x1080 oder höher
   - Dateigröße: < 2MB für schnelles Laden

4. **Nach dem Hinzufügen:**
   - Das Bild wird automatisch als Hintergrund verwendet
   - Falls das Bild nicht gefunden wird, wird ein CSS-Gradient angezeigt

## Aktueller Status

Das CSS ist bereits konfiguriert, um das Bild zu verwenden. 
Sobald `login-background.png` in diesem Verzeichnis liegt, wird es automatisch geladen.
